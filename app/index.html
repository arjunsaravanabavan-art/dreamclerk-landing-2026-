how it works — 8 steps

  1. apply — 90 seconds
  2. interview — 20 minutes, scored on a 4-point rubric
  3. offer — within 48 hours of interview
  4. onboard — first company, first tech lead, first ticket
  5. 5 sprints of pull requests, in a real codebase
  6. capstone project, reviewed by a real engineer
  7. review round — ai pushback, you defend
  8. signed certificate — verified work record

the workspace — what you ship with

  • monaco editor — the same engine that powers vs code
  • sandboxed terminal — real bash, no shared state
  • docker microvm — run node, python, java, go safely
  • jupyter notebooks (jupyterlite · wasm) for ml & data
  • sql playground — sandboxed postgresql per student
  • ai code reviewer — line-level feedback
  • pr flow — submit code, ai tech lead merges or requests changes
  • git client — branch, rebase, merge, all in browser
